# bitcoin-crc32c

`bitcoin-crc32c` is a Rust crate designed to efficiently compute the CRC32C checksum, which is commonly used as a data integrity check algorithm in network communication and storage systems. This crate provides an implementation optimized for different CPU architectures, namely x86_64 and AArch64, leveraging specific instruction sets like SSE4.2 and ARMv8's CRC32 extensions.

And use it in your project:

